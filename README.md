# fullstack-nodejs-jaqueline

Projeto de cadastro de clientes que poderá conter muitos contatos associados. 

## Instalação

npm install

## Uso

npm start ou npm run dev

## Configuração

npm run typeorm:generate

npm run typeorm:run
